🗺️  HOW TO FIX IN XERO
────────────────────

1. 📊 Go to: Reports → Account Transactions
2. 🔍 Select account code (e.g., 494 for Travel International)
3. 🎯 Filter: "Unreconciled" only
4. ✏️  Click "Edit" on first transaction
5. 🔧 Change "Tax Rate" dropdown
6. 💾 Click "Save"
7. ➡️  Move to next transaction
8. 🔁 Repeat for batch

📖 TAX RATE CHEAT SHEET
────────────────────────

Xero Label                    = Code
────────────────────────────────────
"GST on Income"              = OUTPUT
"GST Free Income"            = EXEMPTOUTPUT
"GST on Expenses"            = INPUT (claimable)
"GST Free Expenses"          = EXEMPTEXPENSES
"BAS Excluded"               = BASEXCLUDED (no GST)
